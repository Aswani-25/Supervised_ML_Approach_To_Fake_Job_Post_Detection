# 🕵️ Supervised ML Approach to Fake Job Post Detection

## 📌 Project Overview

This project is a **Machine Learning-based Fake Job Detection System** that identifies whether a job posting is **Real or Fake**.

It helps job seekers avoid scams by analyzing:

* Job descriptions
* Job links (web scraping)
* Suspicious patterns and keywords

---

## 🚀 Features

* 🔎 Manual Job Description Analysis
* 🌐 Job Link Analysis using Web Scraping
* ⚠️ Suspicious Keyword Detection
* 📊 Fraud Risk Score & Recommendation
* 📈 Dataset Evaluation with Metrics
* 📉 Confusion Matrix & Graph Visualization
* 🖥️ Interactive UI using Streamlit

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy
* BeautifulSoup
* Matplotlib & Seaborn

---

## 🤖 Machine Learning Model

* Supervised Learning Approach
* TF-IDF Vectorization
* Model trained and evaluated using Google Colab
* Best performing model saved as:

  * `model.pkl`
  * `vectorizer.pkl`
